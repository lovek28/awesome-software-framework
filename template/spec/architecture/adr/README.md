# Architecture Decision Records (ADRs)

> **Claude:** Add one file per significant decision (e.g. `0001-why-prisma.md`, `0002-api-versioning.md`). Format: Context, Decision, Consequences. Keeps the project self-documenting.

## Template (copy per ADR)

```markdown
# ADR-XXXX: Title

## Context
What is the issue or decision?

## Decision
What was decided?

## Consequences
What are the trade-offs and follow-ups?
```
